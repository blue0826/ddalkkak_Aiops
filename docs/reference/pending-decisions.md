# 미결 결정 추적 (CEO 확인 대기)

> 멀티세션 대장 허브 룰 2 — 대장이 CEO에 올린 확인/결정 요청 중 **답 안 온 것**을 여기 유지하고,
> 매 보고 말미에 재상기한다. CEO가 답하면 이 목록에서 제거하고 결정을 기록한다.
> 대장이 임의로 결정하거나 조용히 넘기지 않는다.

## 대기 중

| # | 항목 | 왜 CEO 결정이 필요한가 | 올린 시점 |
|---|---|---|---|
| 1 | **SCP 모니터링 에이전트 설치 여부** | 현재 에이전트리스(Basic) 수집이라 샘플이 수 시간 간격 + CPU/메모리 수준만 가능. VM에 에이전트를 설치하면 분 단위 + 디스크·네트워크·프로세스까지 관제 가능. **고객사 서버 작업 + 과금 정책 확인 필요.** | 2026-07-20 |
| 2 | **Cloud Logging 권한 확보** | 토글은 켰으나 SCP가 403(권한 없음). 삼성SDS에 monos 키의 **ServiceWatch / Logging&Audit 권한 부여** + **Log Group·Trail 생성** 요청 필요. 이게 없으면 로그 메뉴는 계속 빈 상태. | 2026-07-20 |
| 3 | **커밋/푸시 승인** | 이번 세션 전체 작업(수정 41 · 신규 188)이 미커밋. 헌법상 **push는 CEO 명시 ack 필수**. 논리 단위로 정리해 커밋할지 승인 필요. | 2026-07-20 |
| 4 | **three.js 의존성 추가 승인** | 3D 토폴로지용 `three` / `@react-three/fiber` 설치는 공유 `package-lock.json` 변경. 팀장 세션이 버전·호환성 보고 후 대장이 CEO 승인 받아 반영. | 2026-07-20 |
| 5 | **유료 LLM 사용 여부** | L4 근본원인 분석은 현재 규칙 기반 템플릿 폴백(과금 0). 실제 LLM API 키를 넣으면 품질↑ 이지만 **과금 발생**. | 2026-07-20 |
